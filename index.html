ATAG One API

Introduction

Get all diagnostic data from your ATAG One thermostat with just one simple command.

A typical message displayed by atag-one looks like this: 

$ atag-one.sh -e <email> -p <p6ssw0rd>
{
    "deviceId": "6808-1401-3109_15-30-001-123",
    "deviceAlias": "CV-ketel",
    "latestReportTime": "2015-11-19 13:46:44",
    "connectedTo": "BCU",
    "burningHours": 41.4,
    "boilerHeatingFor": "-",
    "flameStatus": false,
    "roomTemperature": 18.1,
    "outsideTemperature": 13,
    "dhwSetpoint": 60,
    "dhwWaterTemperature": 32.3,
    "chSetpoint": 1.4,
    "chWaterTemperature": 18.1,
    "chWaterPressure": 1.5,
    "chReturnTemperature": 18.1
}

Quick start

First, install Java [ Windows / Debian / Raspberry Pi ].

Installation for Debian 8 Jessie:

# sudo apt-get update && apt-get -y install default-jdk

When successfully installed, java -version should display something like this: 

$ java -version
java version "1.8.0_60"
Java(TM) SE Runtime Environment (build 1.8.0_60-b27)
Java HotSpot(TM) 64-Bit Server VM (build 25.60-b23, mixed mode)
